# Color Mapping for Brand Update

## Text Colors
- `text-yellow-900` → `text-[color:var(--deep-soil)]`
- `text-[#3A2E1F]` → `text-[color:var(--text-warm-dark)]`
- `text-[#5C4A33]` → `text-[color:var(--text-warm-medium)]`
- `text-[#7A674C]` → `text-[color:var(--text-warm-light)]`
- `text-[#A08960]` → `text-[color:var(--text-warm-lighter)]`
- `text-[#C0A987]` → `text-[color:var(--text-warm-lightest)]`

## Background Colors
- `bg-yellow-50` → `bg-[color:var(--soft-cream)]` or `bg-[color:var(--sun-glow)]`
- `bg-yellow-100` → `bg-[color:var(--sunflower-gold)]/30`
- `bg-yellow-400` → `bg-[color:var(--sunflower-gold)]`
- `bg-yellow-500` → `bg-[color:var(--honey-gold)]`
- `bg-white` → `bg-[color:var(--soft-cream)]` or keep as `bg-white`

## Border Colors
- `border-yellow-100` → `border-[color:var(--border-soft)]`
- `border-yellow-200` → `border-[color:var(--border-soft)]` or `border-[color:var(--sunflower-gold)]/30`
- `border-yellow-300` → `border-[color:var(--sunflower-gold)]/50`

## Shadows
- `shadow-sm` → `shadow-[var(--shadow-gentle)]`
- `shadow-md` → `shadow-[var(--shadow-soft)]`
- `shadow-lg` → `shadow-[var(--shadow-medium)]`
- `shadow-xl` → `shadow-[var(--shadow-large)]`

## Border Radius
- `rounded-lg` → `rounded-xl` (12px)
- `rounded-2xl` → `rounded-2xl` (24px)
- `rounded-xl` → `rounded-xl` or `rounded-2xl`
